# FoodShare Frontend

Food donation platform frontend built with:
- **React 18** - Modern UI framework
- **Vite** - Lightning-fast build tool
- **Tailwind CSS** - Utility-first CSS
- **Framer Motion** - Smooth animations
- **Zustand** - State management
- **Axios** - HTTP client

## Setup

```bash
npm install
npm run dev
```

## Features

- Cinematic hero with smooth animations
- Role-based dashboards (Donor/NGO/Admin)
- Real-time donation browsing and requests
- Glass morphism UI with warm color palette
- Fully responsive design
- Loading animations and smooth transitions

## Structure

- `src/pages/` - Page components
- `src/components/` - Reusable components
- `src/services/` - API integration
- `src/hooks/` - Custom React hooks
- `src/styles/` - Global styles and animations
